# Contributing

Thanks for helping improve TapForge Simulator.

1. Fork the repository and create a focused branch.
2. Keep game decisions server-authoritative and validate all remote requests.
3. Test changes in Roblox Studio with at least two simulated players when networking is involved.
4. Open a pull request explaining the player-facing change and how you tested it.

Please keep discussions welcoming and constructive. Bug reports should include reproduction steps and screenshots or video when useful.
