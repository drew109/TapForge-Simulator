# TapForge Simulator

An open-source Roblox simulator starter built with Luau and [Rojo](https://rojo.space/). Tap to earn energy, purchase permanent-in-run multipliers, and rebirth for stronger future runs.

## What is included

- Server-authoritative tapping, upgrades, and rebirths
- Basic anti-spam rate limiting
- Persistent player data with periodic autosaves
- A responsive UI created entirely from Luau
- Rojo project structure that is easy to extend

## Run locally

1. Install Roblox Studio, Rojo, and the Rojo Studio plugin.
2. From this folder, run `rojo serve`.
3. Open a new place in Roblox Studio and connect the Rojo plugin.
4. Press **Play** to test.

To test saving in Studio, publish the place and enable **Game Settings → Security → Enable Studio Access to API Services** only for a safe test place.
